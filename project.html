My Projects

T20 Score Predictor

The T20 Score Predictor is a machine learning project designed to predict the final score of a T20 cricket match based on various match conditions. Using historical match data and advanced regression algorithms, the model provides a highly accurate score prediction.

  • Data-Driven Analysis: Uses match data such as current runs, overs, wickets, and venue.
  • Feature Engineering: Focuses on current run rate, match conditions, and wickets.
  • Machine Learning Algorithms: Linear Regression, Random Forest, Gradient Boosting.
  • Technologies: Python (Pandas, TensorFlow, Scikit-learn), Matplotlib, Seaborn.
